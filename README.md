# TITLE 
User Management System with User Authentication, Role Management, 
and Image Capture 🚀

# INTRODUCTION 
The User Management System is a comprehensive web application developed using React.js for the frontend, Node.js for the backend, and MySQL as the database. This system offers robust features including user authentication, role management, and an image capturing app, providing organizations with a secure and efficient solution to manage employee information.

# FEATURES 🌟
1. User Authentication<br>
  Secure user authentication using JSON Web Tokens (JWT).<br>
  login functionality with password encryption.

2. Role Management<br>
🛡️ Two roles are developed one is Employee or user role and other is Admin role.<br>
🔑 Role-based access control to restrict unauthorized actions.

3. Employee Information Management<br>
➕ Add, view, edit, and delete employee records.

4. Image Storing<br>
🖼️ Upload and manage employee images securely.

5. Dashboard<br>
Created a dashboard for admin to manage employee by edit and delete features.<br>
Visual representation of key employee metrics.<br>
Admin can manage category by adding more category for employees.<br>

# Technologies Used 💻
## Frontend:
React.js<br>
Axios for API calls<br>
Bootstrap<br>

## Backend:
Node.js<br>
Express.js for server setup<br>
JSON Web Token (JWT) for authentication<br>
multer for image store<br>

## Database:
MySQL for data storage<br>
Store data in the table using MY SQL workbench.<br>
Added default password in the table for admin.<br>

# Installation 🚀

## Clone the repository:
git clone https://github.com/NamanPahariya/userManagement.git

### Install dependencies for both the frontend and backend:

## Install frontend dependencies
cd Front-End-Folder<br>
npm install<br>
npm run dev<br>

## Install backend dependencies
cd Server-Side<br>
npm install<br>
npm start<br>

## Set up the database:
Connect the SQL Workbench with the SQL password with your project.<br>
Create a Table for Admin,Employee and category.<br>
Insert the values in Admin,Employee and category table.<br>

## Run the application:

### Start the backend server
cd Server-Side<br>
npm start<br>

### Start the frontend development server
cd Front-End-Folder<br>
npm run dev<br>
